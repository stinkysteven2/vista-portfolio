# Backlog & Definition of Done — IT Showcase

## Definition of Done

Een user story is **klaar** wanneer:

- Alle acceptatiecriteria uit de user story zijn vervuld
- De code is getest (unit tests met minimaal 70% coverage)
- De code is gereviewd en goedgekeurd door minimaal één teamlid
- Alle bestaande tests slagen nog steeds
- De code voldoet aan de stijlgids en best practices van het project
- Documentatie is bijgewerkt (code comments, API docs indien van toepassing)
- Er zijn geen kritieke beveiligingsproblemen aanwezig
- Het responsive design is getest op mobiel, tablet en desktop
- De functionaliteit is toegankelijk (WCAG 2.1 AA)
