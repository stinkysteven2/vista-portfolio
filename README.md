# Software Tester Portfolio

Welkom bij mijn portfolio voor de **Software Tester** opleiding. In dit portfolio documenteer ik mijn voortgang, testwerk en leerervaringen gedurende de module.

## Over de opleiding

Deze module leert de fundamenten van software testen en het maken van professionele testdocumentatie. Software testen is een essentieel onderdeel van softwareontwikkeling — het zorgt ervoor dat software werkt zoals bedoeld, gebruiksvriendelijk is en betrouwbaar functioneert.

Als praktijkvoorbeeld wordt de website **IT Showcase** gebruikt, zodat er direct ervaring wordt opgedaan met het testen van een echte applicatie.

## Leerdoelen

Na het voltooien van deze module ben ik in staat om:

- Zelfstandig **testplannen** te maken
- **Testen** systematisch uit te voeren
- **Testrapporten** te schrijven
- **Verbetervoorstellen** te formuleren en te presenteren
